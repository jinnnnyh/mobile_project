* 2024.09.03 ~ 2024.09.20

### Back-end
* Java17

* MySQL

* AWS(배포)
### Front-end
* Kotlin
* Retrofit
### 개발환경
*InteliJ IDEA
*Android Studio
### ETC
* Git Hub
* Source Tree

* 일정 생성 및 디데이 확인

* 사진 촬영 및 저장
